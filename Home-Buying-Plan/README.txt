Stitch Debt Tracker (Updated)

How to run locally:
1) Open js/firebase-init.js and paste your Firebase config.
2) From this folder, run:
   python -m http.server
3) Open http://localhost:8000

Pages:
- index.html (Home)
- debts.html
- advances.html
- milestones.html
- profile.html

Firestore collections used:
- debts
- cashAdvances
